# filterMaybe

Defina as função

```hs
filterMaybe :: [Maybe Int] -> [Int]
```

Dado um vetor de `Maybe [Int]`, filtre apenas os valores que são formados por Just

```hs
filterMaybe [Just 5,Nothing,Just 7,Nothing] == [5, 7]
```
