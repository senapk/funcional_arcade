# Bubble Sort

- Entrada: Lista ordenável u
- Saída: Versão ordenada de u pelo método de ordenação em bolhas

```hs
bubble [] == []
bubble [1] == [1]
bubble [1,3,4] == [1,3,4]
bubble [4,3,1] == [1,3,4]
```
