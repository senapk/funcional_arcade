# Somar dois números

Escreva uma função soma que recebe dois parâmetros e devolve a soma dos dois parâmetros.

```hs
soma2 2 6 == 8
soma2 -4 6 == 2
soma2 -1 -1 == -2
```

Note que no GHCI, você precisará colocar os valores negativos entre parênteses
`soma 4 (-4)`.
